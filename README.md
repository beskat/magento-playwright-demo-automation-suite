# Playwright TypeScript Cucumber Test Automation Project for Magento

## Project Overview

This project is a demo test automation framework using Playwright, TypeScript, and Cucumber.

## Setup

1. Clone the repository:

```bash
   git clone https://github.com/beskat/magento-playwright-demo-automation-suite.git
   cd magento-playwright-demo-automation-suite
   ```

2. Install dependencies:

 ```bash
   npm install
   ```

## Running Tests

To run the tests, use the following command:

 ```bash
   npm test
   ```
